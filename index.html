How to Generate Battery Report on Windows

To generate the battery report:

  1. Open Command Prompt as an administrator.
  2. Type powercfg /batteryreport and press Enter.
  3. This will generate a battery report and save it as an HTML file, usually in your user directory (e.g., C:\Users\[YourUsername]\battery-report.html).
  4. Open the HTML file to view detailed information about the battery's health, including its original capacity versus its current capacity.
